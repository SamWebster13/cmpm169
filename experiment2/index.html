Experiment 2 - Living Impressions

Description

This generator creates a scrollable mountain scape with the early sunrise over steep and jagged cliffs, shading them in the early dawn mist.

Technical

The scene is built using layers of noise, each part of the screen representing a strip of jagged rectangles, similar to torn paper. The closer the mountains are, the more jagged and darker they appear, creating a sense of depth and distance.

Reflection

I worked on the code alone and faced several challenges in achieving both the main and stretch goals. I wanted to create parallax scrolling for the mountains and a more detailed sun, but due to time constraints, I had to focus on getting the core functionality working first.
